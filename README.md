Electronics Shop Web Project

This is a front-end web application for an Electronics Shop, created as part of a web development course. The project demonstrates skills in HTML, CSS, and JavaScript, including responsive design, product display, and user interface interactivity.

Features

Displays a catalog of electronic products with images, descriptions, and prices

Responsive layout suitable for desktop and mobile devices

Interactive elements such as product cards, hover effects, and buttons

Clean and modern design inspired by real-world e-commerce websites

Easy to extend with backend or API integration in the future

Technologies Used

HTML5 – Structure and content

CSS3 – Styling and layout

JavaScript – Interactive functionality

Optional: Google Fonts, Font Awesome for icons

How to Use

Clone the repository:

git clone https://github.com/yourusername/electronics-shop.git
cd electronics-shop


Open index.html in your browser.

Browse products, view details, and explore the interactive UI.

Project Purpose

Practice front-end web development and responsive design

Build a portfolio-ready project demonstrating HTML, CSS, and JS skills

Simulate a real-world e-commerce website interface
